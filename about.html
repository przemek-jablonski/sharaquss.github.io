Przemysław Jabłoński



Hey there, I'm Przemek, a Mobile Developer based in Cracow, Poland. Currently studying Applied Computer Science at the AGH University of Science and Technology in Cracow (last semester).


Over the years, I've been developing various solutions and cool bits of code in different technologies, starting with developing videogames in C++ (SFML[link]) and Java (LibGDX[link]), through some web development doodles (PHP/Symfony2, Python/Django, JavaEE/Spring[LINKZ!]) and finally coming full-time to Mobile Development, mostly native work with Android SDK.


My expertise is gravitating around Modern Android Development techniques and good practices, including using Modern Android Architecture (MVP, MVVM, Dependency Injection, Liskov Substitution Principle), having practical experience with most revolutionary Android libraries nowadays (eg. Dagger2, Retrofit, Gson, Realm.io), TDD and Testing in Android (Mockito, JUnit, Espresso) as well as knowledge of smaller utility libraries (ButterKnife, Picasso) and official Support Libraries from Google.

This workflow resulted in nice, clean and readable code which you can find apparent in source code of Adaptive Weather[liink] and Betterfy[liink].


Since the beginning I strongly believe in educational aspect of programming, that's why you can see, examine, fork and modify (almost) all of my source code on Github, for free, on most open licenses possible. That's why I am also actively attending local Google Developers Group meetups, hackathons and prelections.
